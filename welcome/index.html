Introduction

FeeSystemAPI is an api developed for an android app called FeeManagemetSystem, this api is based on php 5.2 and works flawlessly.


Requests

Here are all requests and required information to use them along with the paramenters and responses.

Student Login:

              /auth/login.php
Parameters

Username, Password

Admin Login:

              /auth/admin.php
Parameters

Username, Password

Forgot Password:

              /auth/forgotpassword.php
Parameters

Username

Check Attendance:

              /student/check_attendance.php
Parameters

User ID

Check Fee:

              /student/check_fee.php
Parameters

User ID

Add Student:

              /admin/add_student.php
Parameters

fullName, username, studentId, isAdvancePaid, courseStatus, contactNumber, paidFee, password, FK_Programme, FK_Batch, joiningDate

Add Attended Sessions:

              /admin/addAttendance.php
Parameters

studentIDs[], attendedSessions[]

Get All Batches:

              /admin/selectBatch.php
Parameters

No Parameters

Get 15 Batches:

              /admin/getBatches.php
Parameters

start_row

Get All Programme:

              /admin/selectProgramme.php
Parameters

start_row

Get Students of Specific Batch:

              /admin/getBatchStudents.php
Parameters

batchID
